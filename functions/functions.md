Functions

We use functions for code reuseablity means we can use the same block of code mulitple  times.

We cant use numbers, Keywords as name of the function

Normal Function :
function HelloWorld(){
    console.log("Hello World")
}
HelloWorld();

// Ways To make Functions :

// 1: Function Expression :  Here the variable name is function only
let x = function(){
    console.log("Variable Function")
}
x();

// 2: Function Statement : The normal way
function y(){
    console.log("Function Statement")
}
y();

// Arrow/Fat arrow function 
let z = ()=>{
    console.log("Fat Arrow Function");
}
z();
